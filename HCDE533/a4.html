04Milling & Copper Cubes

Cubic Cent

This week I learned a new machine and software combo and how to hone the tolerances to exactly what is needed. The goal was to create a box with box joints that firmly holds itself together.

Poorly made software continued to be a blocker and a time sink this week. Rhino has an incredibly steep learning curve while Bantam is simply an all around terrible experience. Of the five hours it took to get to a solid cube, three of them were spent learning/troubleshooting what is actually a very simple process.

Once I was cutting, it took eight test cuts to get the tolerances within 0.001mm (1μm). I found that the difference between 5μm made a sizable difference in fit. This was, by a large margin, the most precise fabrication technique I have worked with so far. Getting the fit so perfect, and repeatable, was super satisfying.

Process

This week didn't produce as visual of a learning process as previous weeks have, but it was just as involved.

Digital Design

I started by creating the profile for each of the cube's sides in Rhino. The cube was made to be 20mm, but after offsetting the profile by 0.055mm for the tongues and grooves to fit together, an exact cube size was lost. If an exact size was desired, I think it could be done by finding the amount the mill removes or by finding the setting that controls tool pathing in relation to the profile line. The initial profile, before offset, would also have to compensate for the offset.

I offset the profile inwardly because when I was offsetting outwardly, the dogbone cuts that I set to be 1mm in diamter were becoming too small for the tooling to cut. Then export to a .dxf file type. Rhino File

In KiCad, it's simply importing the .dxf file and plotting it to be transferred to Bantam. KiCad File

Bantam is where settings specific to the Othermill can be changed, though good luck using this software. It's so poorly made that I couldn't even see the d-pad for moving the cutter head when setting the touch-off point. I had to remove my FR1 from the table so the tool would be unobstructed. Bantam File

Fabrication

My first several cuts were simply to get the fit of my pieces just right. At first, I altered the offset in Rhino based on measuremnets taken with calipers, but the last three or four of my adjustments were based on the feel of them fitting together.

Once I got the fit right, it was just a matter of cutting my cube once and assembling it.

Now I'm thinking about how I can use what I learned to make something awesome like a 3D PCB or just a simple circuit that's built into the object.
